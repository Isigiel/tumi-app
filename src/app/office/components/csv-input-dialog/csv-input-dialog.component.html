Import Excel Data

  1. Bring your data in the right format

    Please make sure that your table has the following structure before continuing

    First Name Last Name E-Mail Faculty Country Program
    Andrea Jung max@mustermann.de Fakultät für Informatik Türkei E
    Required for all submissions Only required to update incoming students
    Additional columns will be ignored. One row is reserved for the table headers. Don't have any more rows than your data requires
    The names of your columns will be ignored, important is the order in which they are arranged in your table
  2. Save your list as a .csv-file

    Use the save as dialog in excel and select the CSV UTF-8 (Comma delimited) (*.csv) filetype and save your list

  3. Open your list in an editor

    Use a code editor or the windows editor to open your list that's saved as .csv now. Make sure to not open it in excel but instead open it as text. On windows this works well by right clicking the file and selecting Editor from the Open with menu

  4. Paste the data into this app

    Now copy all the data ( you could use CTRL+A and CTRL+C to do so) and paste it into the text area below
